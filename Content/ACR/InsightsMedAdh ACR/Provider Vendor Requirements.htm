Provider Vendor Requirements

MA.100: The provider vendor application shall display a visual indicator when the worklist summary response API alert value is greater than zero or an item in the worklist response API priority is equal to 5.

MA.101: The provider vendor application shall identify, per patient, the worklist task count and/or the task names adjacent to the Med Adherence Summary's lowest PDC score, units and disease class.

MA.102: The provider vendor application shall display a unique visual indicator based on the value of the adherence summary response API pdcWorstStatus.
