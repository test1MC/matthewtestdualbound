Common Message Header/UIB

For EDIFACT users: The fields in the XML header shall*106R-023 match those in the EDIFACT – UIB Header

<MessageID>

Each message sent from a participant, including Error and Status, shall*106R-024 have a unique Message ID.

<SentTime>

The date and time the message was sent. Shall*106R-026 be sent in UTC coordinates only.
