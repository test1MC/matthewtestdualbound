Cancel Process

Cancelling a process sends a PACancelRequest to the PBM/payer, which begins the process of cancelling an ePA case.

Cancel Process Using the ProcessId

Request Template

[PUT] {{host}}/cancel/processes/{processId}

Request Parameters

M/C – Mandatory or Conditional

Parameter

Data Type

M/C

Description

processId

Long

M

Unique Surescripts ID which represents the process instance.

Request JSON Fields

M/C – Mandatory or Conditional

JSON Field

Data Type

M/C

Description

user

an..250

M

ID of the end user logged into the provider vendor system.

cancelCode

an2

C

NCPDP code for the reason ePA is being cancelled.

Values:

AC=Prescriber is no longer responsible for patient’s care

BW=Change to a different medication

BY=Other

isPatientNotified

an1

C

Boolean value signifying if the patient was notified.

Values:

Y

N

note

an..2000

C

Free text note to PBM/payer regard the cancellation of an ePA.

Request Example

[PUT] https://[host]/uiservice/api/v0/cancel/processes/1234

{  

    User: “Jake”,

    cancelCode: "BW",

    isPatientNotified: “N”,

    note: “Sample note”

}

Cancel Process Using InitiatorsProcessId

Use the InitiatorsProcessId to cancel a process.

Request Template
