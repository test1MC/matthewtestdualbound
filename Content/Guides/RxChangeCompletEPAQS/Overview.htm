About This Guide

The Surescripts RxChange for CompletEPA® QuickStart™ Supplement Guide was created to assist pharmacies in automating the retrospective prior authorization process through the use of the RxChange message.

This supplement is to be used in addition to the Surescripts Prescription Routing 10.6 Implementation Guide.
