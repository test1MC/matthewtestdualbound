Key Elements of the RxChangeResponse

This section outlines some of the key elements on the RxChangeResponse for the prior authorization workflow. Surescripts sends these fields to the pharmacy to provide visibility into the outcome of the prior authorization. For XML documentation, refer to the schema, SS_SCRIPT_XML_10_6MU.xsd.

Element Name

Comments
Medication Prescribed

PriorAuthorization

 

  Value

Provider vendor systems should send the prior authorization number, if provided by the PBM/payer.

  Qualifier

G1 = Prior Authorization Number
