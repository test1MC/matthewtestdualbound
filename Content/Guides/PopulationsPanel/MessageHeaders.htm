Message Headers

Overall File Header (Surescripts XML Schema)

Surescripts uses an overall file header for each file, and every file contains a set of messages. Each message contains a common message header, which is described further in Common Message Header (NCPDP XML Schema) on page 1.

Common Message Header (NCPDP XML Schema)

Surescripts uses a common message header for all messages. For Medication History Panel Management, a common message header is included in each patient’s RxHistoryResponse. The common message header, also called the XML wrapper, follows the NCPDP SCRIPT standard version 10.6 with one addition; the Sender Software Version, which was added in the NCPDP SCRIPT standard version 10.7. For this implementation guide the digital signature is not used.
