Message Examples

The Mailboxing service examples found in this section are provided in addition to those found in the NCPDP SCRIPT Standard Examples Guide.

Examples Disclaimer

Examples provided throughout this Guide are not intended to be all-inclusive. This pertains to example workflows, element-specific (field) examples, or message examples. Customers should not restrict coding to the examples used herein.

When developing, this Guide should be used along with the accompanying schema file, additional documentation referenced, and applicable customary XML coding standards.

The following examples use the NCPDP version 20170714 which is a republication of version 2017071.

GetMessage Sequence Example

This section contains the following GetMessage sequence example.

 

  • (1) First GetMessage Sent by Pharmacy to Their Mailbox
  • (2) Mailbox at Surescripts Responds with a NewRx Message
  • (3) Second GetMessage Sent by Pharmacy to Mailbox
  • (4) Surescripts Responds to Pharmacy with a RxRenewalResponse
  • (5) Third GetMessage Sent by Pharmacy to Their Mailbox
  • (6) Surescripts Responds with a Status 002 - Mailbox is Empty

The GetMessage is highlighted in the message body.

    <SentTime>2018-07-04T19:14:15.6Z</SentTime>

      <UsernameToken>

        <Username>mailboxname</Username>
