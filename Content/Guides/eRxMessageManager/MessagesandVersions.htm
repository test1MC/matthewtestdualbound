Messages and Versions

This pilot supports version 10.6MU and version 6.0 (6.0 supports the new SCRIPT version 2017071) and the following NCPDP messages:

  • NewRx (including electronic prescribing of controlled substances (EPCS)
  • RefillRequest and RefillResponse for 10.6MU, or RxRenewalRequest and RxRenewalResponse for 6.0
  • RxChangeRequest and RxChangeResponse
  • CancelRx and CancelRxResponse
  • Status
  • Verify
  • Error

Customers who are using the original version of 10.6 (pre-10.6MU) will need to use the Certification & Testing Suite for message validation. Future versions of eRx Message Manager will support additional products and transactions.

RefillRequest and RefillResponse are renamed RxRenewalRequest and RxRenewalResponse in version 6.0.
