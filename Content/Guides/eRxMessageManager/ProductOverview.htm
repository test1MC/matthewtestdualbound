Product Overview

About eRx Message Manager

The Surescripts eRx Message Manager is used for testing in the staging environment prior to certification for production. If you have questions concerning this Guide, please direct them to your Surescripts Account Manager.

Any examples provided in this User Guide do not include actual patient data. Messages sent to the Surescripts Staging environment must use fictional patient data and cannot contain Protected Health Information (PHI).

About this Guide

This Guide was created to assist Surescripts customers in the use of the eRx Message Manager to trigger messages that each would expect to receive in a production environment. This Guide addresses the tools and functionality, impersonation of message receiver, selection of response messages, substitution of real patient data, and the setup and execution of test cases.

The eRx Message Manager allows a user to confirm that the vendor software can send syntactically valid messages and process incoming messages. It can also be used for product demonstrations or training.

Intended Audience

The intended audience of this document is anyone responsible for testing the ability to send or receive messages and to verify how those messages will interact with the vendor’s software.

Access

User access permission is configured in Admin Console by a Surescripts resource or by an individual with user management access.
