STS-010

STATUS TYPE CODE

<Code>

an..3

Status

010 – Successfully accepted by ultimate receiver

Error

Codes used to relay rejected communications.

600 = Communication problem – try again later

601 = Receiver unable to process – do not retry

602 = Receiver System Error – try again later

900 = Transaction rejected

Note: When receiving Error Detail: 900: ASSOCIATED ELIG TXN NOT FOUND, the error may be due to a delay in the Surescripts system registering the eligibility response. Customers will need to implement a delay of at least 10 seconds between the eligibility response and the medication history request to avoid this error. If this error is received, the customer should resend the medication history request.
