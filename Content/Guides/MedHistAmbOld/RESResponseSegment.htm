A “Denied” (D) response must be accompanied by Free Text (RES-040) since there is not a valid code list qualifier.

Values:

A = Approved

D = Denied. Use the denied code for a scenario where data cannot be released for a patient due to contractual or business constraints. This code should not be used to represent processing or syntax errors, instead an Error message should be sent.
