SIG – Structured Sig Segment

The Structured SIG Segment is a conditional segment. The intent is not to facilitate the reconstruction of the SIG to human readable form (English), but rather to communicate through electronic means the SIG components in a controlled, well-defined structure. It is to be used to relay information between the prescriber and the pharmacist. It is not patient instructions.

SIG data will only be sent in fill data.

SIG Free Text String Indicator (030-01) contains important instructions affecting the creation of the Structured SIG Segment and its relationship to the DRU Dosage Segment (030-02 or 030-03) SIG instructions. Since this is a conditional field, the message should be able to be processed using the free text instruction field in the DRU segment for those participants not supporting the structured SIG segment.

Required when prescriber specifies a hard stop.

SIG-030

FREE TEXT

<FreeText>

   Used to reflect the text string express of the Sig. It should always be used, and in addition is the only composite to place a free text Sig from a generating system that CANNOT generate a structured Sig.

M

SIG-030-01

Sig Free Text String Indicator

<SigFreeTextStringIndicator>

n1

Indicates the system capability of representing the instructions.

Values:

1 = Capture what the prescriber ordered. The Structured Sig Segment free text is used - free text is what the prescriber selected. DRU Dosage (030-02 or 030-03) SIG instructions must contain the same text. The Structured Sig Segment contains the codified Sig fields from the prescriber’s order.

2 = Reconstructed from structured Sig. The Structured Sig Segment free text is used and it is a direct translation of codified Sig values and must not be structured English language or pharmacist/patient friendly. DRU Dosage (030-02 or 030-03) Sig instructions free text contains what the physician selected.

3 = Pure free text. Value 3 is not used in the SCRIPT Standard. If the structured Sig cannot be generated, the segment is not used. The DRU Dosage DRU-030-02 or DRU-030-03) Sig instructions free text is used. DRU Dosage DRU-030-02 or DRU-030-03) Sig instructions free text contains what the physician selected.
