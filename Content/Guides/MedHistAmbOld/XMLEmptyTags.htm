XML Empty Tags

Participant shall*106M-005 not send empty elements within XML messaging with exception of the RxHistoryResponse-<Approved> element.

All non-essential characters shall*106M-002 be suppressed from the message where permissible (see EDIFACT Representation on page 1). Non-essential characters include leading spaces, trailing spaces, leading zeroes, and trailing zeroes where permissible (see Numeric Representation on page 1).
