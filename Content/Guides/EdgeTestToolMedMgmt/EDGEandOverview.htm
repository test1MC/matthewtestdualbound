Edge Test Tool

Edge Test Tool Overview

Edge is a testing tool to assist customers and integration teams in connecting to Surescripts. The Edge Test Tool connects to the edge of our network and can give our vendors an edge when implementing our APIs. Medication Management for Adherence has a hub and spoke model connecting PBMs to EHRs and this tool is intended to provide simple message sending / receiving that can be used by the vendors that connect to our network.

Any examples provided in this guide do not include actual patient data.

Pre-Log In Setup

Users must be set up with a Surescripts Admin Console username and password in order to access the Edge Test Tool. Once set up with Admin Console credentials, users can log into the Edge Test Tool for the purposes of testing Medication Management for Adherence FHIR® messaging.

For issues with logging in or obtaining an Admin Console username and password, please contact Surescripts Support or your Surescripts Account Manager.

First Time Log In

The first time a user logs in they are presented with a screen that allows them to generate a test NPI and SPI for purposes of sending Medication Management for Adherence FHIR® messages within the Edge Test Tool.

Fill out all of the required fields and click the Generate Test NPI button to assign an NPI/SPI to the user account. Once created, the user is unable to be edit their information without administrator assistance.

Logging In

Login to the Edge tool with your Admin Console Username and Password.
