Entering Send Parameters

Refer to the STAGE test plan for the values to be used for the Sender UID, Username and Password.  These values are used for authentication when the FHIR® message is Posted to the Mailbox.

Patient Member Id

 

Patient Member Id will override the Patient.Identifier.value@value included in the FHIR® message loaded. Leave this text box blank, for STAGE testing we will not override the Patient Member Id.
