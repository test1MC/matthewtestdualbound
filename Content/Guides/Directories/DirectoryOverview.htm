Directory Overview

About Directory

Surescripts maintains organization and provider directories for message routing. Proper management and maintenance of records by customers is critical to successful message exchange. Customers rely on the information maintained by other customers to find the appropriate communication paths.

The directory is a resource used in conjunction with the customer's implemented product(s).

The Surescripts Directory Messaging Interface is the primary method for customers to perform the task of directory administration. It allows for single-source transaction flow and automation of directory maintenance through a set of messages.

Applications using Surescripts Directory Messaging can be implemented on any platform, in any programming language, as long as the application can create and consume messages defined for the service interface. For directory message examples, please see the Examples section.

Directory maintenance and administration tasks can also be performed using the Surescripts Admin Console.

Admin Console

The Admin Console is a tool that provides visibility into the directories and messages that transmit across the Surescripts network as well as the latest capabilities and validations available for Surescripts Directory information. The organization and provider directories that are accessible within the Admin Console provide the foundation for the transactions and also provide vendor customers with the necessary tools and resources to maintain updated directory listings within their respective applications.

Please contact your Account Manager for an introduction to the Admin Console and for training.

About This Guide

This Guide illustrates the use of Surescripts Directory Messaging, which allows for updates to the directory information, as well as the ability to download aggregate copies of the data. Updates may consist of adding new providers and organizations, making changes and deactivating the directory entry of providers and organizations.

The terms "message" and "transaction" are used interchangeably throughout this Guide.
