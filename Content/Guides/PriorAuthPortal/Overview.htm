About This Guide

The Surescripts CompletEPA® QuickStart™ User Guide was created to assist providers in initiating the prior authorization process.

The intended audience of this document is any user that will have direct interaction with the application. This may include users responsible for initiating, tracking, and monitoring the status of prior authorizations.

Any examples provided in this user guide do not include actual patient data.

Application Overview

CompletEPA® QuickStart™ enables providers to submit fully electronic prior authorization requests right away, even if electronic prior authorization is not yet available through the provider vendor system. CompletEPA® QuickStart™ is a free, easy to use, web-based solution with no application to install and can be accessed at www.surescripts.com/quickstart.

Using CompletEPA® QuickStart™

CompletEPA® QuickStart™ allows users to initiate prior authorizations using two methods:

  • Initiate a prior authorization using a TRX Code provided by the pharmacy
  • Initiate a prior authorization by entering patient and medication details within the application

What is a TRX Code?

The TRX Code is sent by the pharmacy to initiate prior authorization after a prescription drug has been rejected at the pharmacy. The code is linked to a specific patient, drug, and benefit information.

The TRX Code is 16 characters long (XXXX-XXXX-XXXX-XXXX). Each code will contain a combination of randomly selected uppercase and lowercase letters and numbers. TRX Codes will not include the following characters: 0's, O's, o's, l, 1, I, i, m, n, 8, B, 2, Z.
