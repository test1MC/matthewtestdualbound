Worklist Overview

The tasks listed on the Worklist are specific to prescriber location where the user is signed in. This assignment is listed in the top right under the “Working at” designation. The location and prescriber can be changed by clicking on the drop-down arrow adjacent to the listed assignment. The Worklist provides a centralized view of all tasks that may require action from the user accessing it. Worklist tasks reflect access assigned to each user – a user is associated with tasks for specific providers/patients as well as assigned a role for each of those affiliations.

  1. Active List Name – Users can easily see which list is currently being presented to them (Worklist or Task History). Within the Worklist view, hovering over the information icon () will present the user with a count of tasks for each product enabled. Priority tasks may be shown within this tooltip, but task priority is currently not supported.
  2. Refresh Button – Click the Refresh button to manually refresh the current list.
  3. Filtering Options – Users can select Filtering Options to show the list of available filters. Currently, users can filter by Patient First Name, Patient Last Name, Patient Date of Birth, Created From/Through Date, and Due Date From/Through.
  4. Task List – Shows the list of active tasks that a user has access to. By selecting a column header (other than Task), users are able to sort their tasks in ascending/descending order. Only one sortable criterion may be active at one time. indicates a descending sort, while indicates an ascending sort.
