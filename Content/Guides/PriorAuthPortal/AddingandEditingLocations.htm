Adding and Editing Locations

Locations allow the user to more easily identify a provider’s location.

When the user completes ID verification, a location will be added to the provider’s profile. To add more locations, follow the steps below.

Adding Location

  1. Click on the drop-down next to the provider’s name.
  2. Select Add Locations.
  3. Fill in location information and click the Save Location button.
  4. Click the Confirm button.

Editing Location Name

To edit the name of a location, click on the name next to the Pencil icon and enter a new value.
