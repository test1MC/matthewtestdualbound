Initiating a Prior Authorization Using a TRX Code

To initiate a prior authorization using an existing TRX Code, the user can either:

  • Enter the TRX Code as part of the sign in process on the home page, or

  • If the user is already logged in, click the Add New PA button on the navigation bar. Enter the TRX Code in the appropriate field and click Submit button.

Upon entering a TRX Code, a task will be added to the user’s Worklist to initiate the Prior Authorization (PA).
