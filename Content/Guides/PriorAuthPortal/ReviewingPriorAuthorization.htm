Reviewing Prior Authorization Task History

A specific process related to a prior authorization case can be accessed by opening the Task History from the top navigation menu. Select a specific prior authorization case from the Task History worklist to open and view these detailed processes.

  1. Process Details – Includes the details of the process, including process type, Status (In Progress or Complete), and Locked By (if the current process is locked by another user – also indicated by a closed lock in the Task/Patient Header).
  2. Case Details – Includes the details of the ePA case including PA Case ID (assigned by the PBM/payer), associated PA Notes, and details of action required.
  3. Process History – Includes task type, status, recipient and responded date.
  4. Completed Question Set Indicator – This button allows users to review the responses to a completed ePA Question Set.
  5. Cancel Process Button – This button is used to cancel an in progress ePA; this will generate a PACancelRequest to the PBM/payer.
