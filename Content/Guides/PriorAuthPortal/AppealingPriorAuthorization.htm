Appealing a Prior Authorization

A PBM/payer may allow a denied prior authorization to be appealed directly from the prior authorization. If enabled by the PBM/payer, the denied prior authorization response will display an option accept the denial or appeal.

  1. PA Case ID – This is the PA Case ID as assigned by the PBM/payer. This may be required if any documents are requested by the PBM/payer or other forms of manual communication are required.
  2. PA Note – This free text box allows a user to indicate why they are appealing the PBM/payer’s response. The PBM/payer may respond with an approval/denial or may request additional information through the use of additional Question Sets.

  3. Appeal Button – Selecting this will allow the user to submit the appeal to the PBM/payer.

  4. Options Drop-Down – The Option’s drop-down menu allows the user to return to the previous list (Worklist) by selection the Go to Worklist option.
