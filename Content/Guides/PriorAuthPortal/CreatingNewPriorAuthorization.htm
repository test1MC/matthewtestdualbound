Creating New Prior Authorization

Prescriber ID Verification Process

Prescriber ID verification is required only for prior authorization requests initiated at the time of prescribing without a TRX code. This one time process can be completed by the prescriber or a delegate working on their behalf by completing the following:

  1. Click the Add New PA button on the navigation bar.
  2. Click the Start Prescriber ID Verification button under Create New Prior Authorization.
  3. If the user is a delegate, enter the prescriber’s information and click the Submit button. If the user is the prescriber, this information will be automatically populated.

    4. Prescriber’s name must match the name under which the NPI is registered.

  4. Select method for ID verification:
    • Fax option allows users to confirm the prescriber’s identification by receiving a fax confirmation code. Select the number at which you’d like to receive the fax confirmation code from the list of fax numbers associated to the NPI to send the confirmation code. After the fax is received, the confirmation code must be entered back into the portal.

      • There may be a short delay in receiving the fax with the confirmation code.

      If fax was selected for the ID verification method, the user must complete the following steps:

      • Click Enter Code button.
      • Enter the prescriber’s Fax Confirmation Code and click the Confirm button.
      • The prescriber will receive a password reset email and will need to accept a terms of use and business associate agreement upon their first log in.
    • Third Party Verification allows users to complete ID verification via a third party processer by answering questions regarding their credit history.

      • This option must be completed by the prescriber and not by a delegate. Delegates will not see this option, but will be able to send a request to the prescriber to verify their account. The prescriber must use their home address and not the address of their practice when answering the survey provided. Once the session has started, the user has 20 minutes to complete the validation.

  5. Enter prescriber’s location information and click Save Location button.
  6. Once these steps are complete, Prior Authorizations may be initiated using the prescriber’s profile.

Starting New Prior Authorization

  1. Click the Add New PA button on the navigation bar.

  2. Click the Start Prior Authorization button.

  3. Search for a patient using one or more of the following fields:
    • First Name
    • Last Name
    • Middle Name/Initial
    • Date of Birth
    • Gender
    • ZIP/Postal Code

  4. Select the applicable patient in the Patient List section and click the Confirm Patient button. If the patient is not available after the search is complete, create a new patient by clicking on the Create New Patient button and filling in the applicable fields.

  5. Once the patient has been confirmed, click on the applicable health plan and PBM/payer name in the Plan section and click the Confirm Plan button.

  6. If a plan cannot be found for the patient, click Create New Plan button to manually enter the Plan/PBM Name, BIN, and PCN.

  7. In the Prescription section, use the type-ahead field to search for the medication. Available formulations will appear in a drop-down list for the user to select.

  8. Fill in the remaining prescription fields and click the Confirm Prescription button.

  9. In the Pharmacy section, search for the pharmacy using one or more of the applicable fields.

  10. Select the appropriate pharmacy and click the Confirm Pharmacy button.

  11. Click the Initiate PA button to send your prior authorization request to the patient’s PBM/health plan. Once the PA has been created, the response will automatically show up on the user’s Worklist to process. The response will include a new task from the PBM/payer with a question set to complete or additional information related to status of the prior authorization.
