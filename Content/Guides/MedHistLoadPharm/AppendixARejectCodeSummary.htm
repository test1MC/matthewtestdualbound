Appendix A: Reject Code Summary

When the Pharmacy Extract File Header field Load Status is “02 – File loaded with errors” or “03 – File contains errors – File not loaded”, the following Reject Code values may be reported.

Reject Code

Explanation

Field Possibly in Error or Example

1001

Required segment missing

 

1002

Required detail records missing

Only a file header HDR and trailer TRL are present. No detail records are present.

1003

Unknown segment

There is an extra blank line in the file.

1006

Required field missing

 

1007

Invalid field length

 

1008

Field value not found in validation table

Used for fields that have a constrained list of valid values when the value supplied is not an acceptable value. For example, gender is constrained to the characters M, F or U.

1009

Invalid character(s) in field

Invalid characters include slash, star, pipe, colon, tilde or a space in a numeric field.

1010

Extra data found after segment

A detail record is included after the trailer record.

1012

Invalid Record Count

The file or list trailer has the wrong value in the record count field.

1013

Invalid Sender/Receiver ID or Password

 

9000

Other Error

Error is not one of the codes, see free text for description.
