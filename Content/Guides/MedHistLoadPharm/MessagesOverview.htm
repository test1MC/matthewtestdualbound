Medication History Load Message Flow

The Surescripts Medication History Load for Pharmacies service assists pharmacies by hosting medication history data using the pharmacy extract file. If the pharmacy is participating in the PDMP (Prescription Drug Monitoring Program) this data is also sent on to the State for reporting purposes.

The file is sent by the pharmacy in a batch file process on a daily basis and loaded into a repository at Surescripts. Surescripts will format and validate the data and, if an error exists, provide the pharmacy with a secure web user interface to allow for correction prior to sending the data out to the individual states or their designees on their behalf.
