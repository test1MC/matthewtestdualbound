Remaining Optional Segments

The ADT message set contains many other optional segments. Surescripts does not require or utilize any of them for Medication History – Acute processing. These segments will not be loaded, viewed, or returned by Surescripts. Surescripts will only validate that the structure is syntactically sound for all segments received.
