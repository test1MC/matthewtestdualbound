Medication History – Acute Request – ADT Message

Introduction

The request process is triggered by events that already take place within a hospital system. The ADT events are the Medication History – Acute request transaction because they occur at the time where a Medication History – Acute response would be valuable. These ADT events also contain the patient demographics required to perform a Medication History – Acute request. 

In the specifications below, note that not all elements and segments are needed or utilized by Surescripts. The segments that are grayed out are optional and do not contain elements that Surescripts requires to process a Medication History – Acute request.
