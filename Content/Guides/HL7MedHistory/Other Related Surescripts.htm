Other Related Surescripts Guides

Prescription Benefit Implementation Guide

This Surescripts Prescription Benefit Implementation Guide was created to assist Pharmacy Benefit Managers (PBMs) and Prescriber Systems in developing and transferring messages needed to provide PBM member data (eligibility information, pharmacy benefit coverage, and group-specific formulary information) to physicians in an ambulatory setting.

The audience for this document includes any participant responsible for developing a system interface for these electronic prescribing transactions. The guide describes the Surescripts Prescription Benefit transaction sets and provides other information needed for implementation.

Directories Implementation Guide

The Surescripts Directories Guide was created to assist pharmacies and Prescriber Systems in developing and transferring directory related messages containing Surescripts related and demographic data for a pharmacy and prescriber. Complete and accurate directories are essential for prescription routing.

The audience for this document includes any participant responsible for developing a system interface with the Surescripts network. This guide describes the directory message sets and provides other information needed for their implementation.
