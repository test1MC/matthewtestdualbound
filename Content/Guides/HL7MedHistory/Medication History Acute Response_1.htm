Medication History –Acute Response – RDS Format

Introduction

The RDS message is defined as a pharmacy/treatment dispense message (Event O01). Surescripts utilizes it to return a Medication History – Acute response in a data element format. The RDS message utilizes an RXD segment that contains the medication-dispensed elements to portray a Medication History response.

Surescripts does not utilize all segments available within the RDS. The table below outlines the segments and elements populated by Surescripts for a Medication History – Acute response.
