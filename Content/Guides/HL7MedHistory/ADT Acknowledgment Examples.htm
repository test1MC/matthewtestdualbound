ADT Acknowledgment Examples

Positive ACK from Surescripts back to the sender.

MSH|^~\&|MEDS|RXHUB|P00000000002002|REQUESTINGSYSTEM^1234567890|200304230823|PASSWORD|ACK^A01|999989|P|2.3|MSA|AA|123456||||0

Surescripts supplied fields:

Element

Element Name

Value

Description

MSH-3

Sending Application

MEDS

Surescripts defined application ID for the sending application.

MSH-4

Sending Facility

RXHUB

Surescripts defined participant ID for the sending application (Surescripts).

MSH-5

Receiving Application

P00000000002002

Identifier of the participant who sent the original request.

MSH-6

Receiving Facility

REQUESTINGSYSTEM^1234567890

Additional identifier for original sender.

MSH-8

Security

PASSWORD

Surescripts password in the sender’s system.

MSH-10

Message Control ID

999989

Unique control number for this transaction.

MSA-1

Acknowledgement Code

AA

Application Accept.

MSA-2

Message Control ID

123456

Message control ID of the original transaction (trace number).

MSA-06

Error Condition

0

Indicates that the transaction was successful.
