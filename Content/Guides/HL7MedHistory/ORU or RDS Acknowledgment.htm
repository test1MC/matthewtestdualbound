ORU or RDS Acknowledgment Example

Positive ACK from participant to Surescripts.

MSH|^~\&|P00000000002002|REQUESTINGSYSTEM|MEDS|RXHUB|200304230823|PASSWORD|ACK^R01|999989|P|2.3|MSA|AA|123456||||0

Surescripts supplied fields:

Element

Element Name

Value

Description

MSH-3

Sending Application

P00000000002002

Surescripts defined participant ID for the sending application (P00000000002002).

MSH-4

Sending Facility

REQUESTINGSYSTEM

Additional identifier for sender.

MSH-5

Receiving Application

RXHUB

Identifier of the participant who sent the original request.

 

 

 

 

MSH-8

Security

PASSWORD

Password to get into the Surescripts system.

MSH-10

Message Control ID

999989

Unique control number for this transaction.

MSA-1

Acknowledgement Code

AA

Application Accept.

MSA-2

Message Control ID

123456

Message control ID of the original transaction (trace number).

MSA-06

Error Condition

0

Indicates that the transaction was successful.
