The following diagram depicts various scenarios where Error messages are sent in response to a Benefit Request that used the RFS or PMBC workflow.

Scenario 1:Surescripts Cannot Recognize the Message. Surescripts returns an error message.

1a) A provider vendor sends a Benefit Request to Surescripts.

1b) Surescripts cannot recognize the message (e.g. improperly formed XML) and sends an error back to the provider vendor.

Scenario 2: Surescripts Finds Message Errors in Benefit Request. Surescripts returns an ERROR message.

2a) A provider vendor sends a Benefit Request to Surescripts.

2b) Surescripts recognizes the format but finds errors (e.g. schema validation error) in the message. Surescripts returns an ERROR message.
