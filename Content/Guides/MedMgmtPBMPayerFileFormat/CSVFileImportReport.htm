CSV File Import Report

The file import report is created by Surescripts when processing the import file from the PBM/payer vendor.  This file contains the results (success or error) of the import process for each message.   In the diagram below, this file is created in Step 2 and the PBM/payer retrieves the report in Step 3.

This file is produced after the import file has been processed. A single report file is created for a given import file. The timing of when the import report file will be available in the from_surescripts is based on the number of messages in the import file.   (#messages x 15 sec) + (10 sec)Example 100 messages:  (100 x 15) + 10 = 1510 sec
