Process Overview

Message Flow

Medication Management for Adherence enables the electronic delivery of PBM/payer medication management messages and member’s medication adherence profile. Through the provider vendor interface, the practitioner views and responds to the PBM/payer medication management messages and accesses the member's adherence profile. The graphic below depicts the transaction flow between customers in this process when the PBM/payer vendor utilizes SFTP file exchange for a pilot.

Message Flow

  1. PBM/payer SFTP pushes a file of messages.
  2. Surescripts processes the messages to create workflow processes and PDCs and puts an import report on the SFTP site.
  3. PBM/payer pulls the import report to review results of the message imports.
  4. Provider vender user accesses Medication Management for Adherence through the provider vendor system to view messages and optionally submit feedback.
  5. Surescripts generates a disposition report after messages have expired based on the cadence at which the PBM/payer sends messages.
  6. The PBM/payer pulls the disposition report.
