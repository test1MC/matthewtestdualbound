Directory Error Messages

The following errors are some of the most common directory-related errors.

Description Codes

900 System error
  A system error occurred when parsing the message.
901 Authorization failed
  Code is sent when a username or password failed authorization.
902

Validation failed
  Code is sent when validating the message type, a description of the validation error will appear in the <Description> element.
903 Duplicate prescriber found.
  Code is sent when sending a AddPrescriber message and an existing prescriber matches the one being sent, a description of the validation error will appear in the <Description> element. The error message will be accompanied by a AddPrescriberResponse message.
904 File not found.
  Code is sent when sending a DirectoryDownload message and the requested file cannot be found.
