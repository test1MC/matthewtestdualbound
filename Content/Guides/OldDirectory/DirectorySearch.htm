GetProvider

Prescriber vendors use the GetProvider message to search for pharmacies by NDPDPID and any organization type using address information.

Pharmacy vendors use GetProvider to search for providers using SPI or address information.

All vendors can search for their data they manage using the appropriate ID or address. Information will be returned only if the network customer has access to the record. If the data supplied, or the structure of message is invalid, an error message will be returned.

Either a GetProviderResponse or an Error will be returned.

GetProviderResponse

A GetProviderResponse will be returned when zero or more results were found matching the given search parameters. In the instance where no results were found, the Response segment will contain a Denied element with a reason indicating that no results were found. See Examples-Get Provider Response for additional examples.
