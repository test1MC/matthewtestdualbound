Observation

The Observation API retrieves the member’s adherence profile (PDC ratios)

Workflow Considerations

It is recommended to show the adherence profile in conjunction with the medication management message detail and questionnaire in areas of the workflow where the provider will take action on non-adherence or gaps in care. 

The diagram below illustrates where the Observation API should be used in a typical patient visit workflow.
