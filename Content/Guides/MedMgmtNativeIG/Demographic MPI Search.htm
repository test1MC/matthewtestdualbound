Demographic MPI Search

This will accept a set of demographics including name, date of birth, gender, zip code.

Demographic MPI Search Template Example

[GET] {{host}}/mmpoc/api/v0/patient-ids?firstName={firstName}&lastName={lastName}&zipCode={zipCode}&birthDate={birthDate}

Request Parameters

M/C - Mandatory or Conditional

Request Parameter

M/C

Description

M

Patient first name.

lastName

M

Patient last name.

middleName

C

Patient middle name.

C

Patient address zip code must be 3-5 characters.

birthDate

M

Valid data format.

gender

C

Values:  M, F, U

[GET] {{host}}/mmpoc/api/v0/patient-ids?firstName=Bob&lastName=Price&zipCode=55044&birthDate=19550524
