Response Message

Responding to a medication management message is strongly encouraged, but is optional. This feedback will be stored in a Surescripts repository and can be queried by PBM/payers at any given time.

Data elements provided to the PBM/payers upon query are the following:

Data Element Name

Description

PBM Message Id

PBM/payer’s unique message Id associated with the medication management message.

Message Type

Message type of HRM, Missing Medication or Medication Adherence.

Patient Id, Patient Name, Patient DOB

Patient information supplied by the PBM/payer.

Received Date

Date the message was received from the PBM and the workflow process was initiated.

Message Viewed Date

Date of the CommunicationRequest(s).

Message Not Viewed Date

If no ComminicationRequest or QuestionnaireResponse is sent before the date the task expires.

Questionnaire Response Date

The date that a response was submitted.

Workflow Process Id

The workflow process Id the PBM/payer will need when acknowledging the receipt of disposition data for a given message.

Workflow Task Id

The workflow task Id the PBM/payer will need when acknowledging the receipt of disposition data for a given message.

Questionnaire Response

The response posted, the SPI and author display value.
