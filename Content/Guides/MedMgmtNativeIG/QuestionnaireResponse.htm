QuestionnaireResponse

The QustionnaireResponse API is used to submit a completed questionnaire back to the PBM for a medication management message (High Risk, Medication Adherence or Missing Medication.  The QuestionnaireResponse endpoint link should be endpoint saved in your system from the CommunicationRequest.  The task Id passed is also obtained from the CommunicationRequest API.

Workflow Considerations

It is recommended to show the questionnaire with the medication management message detail and the adherence profile in areas of the workflow where the provider will take action on non-adherence or gaps in care. 

  • The CommunicationRequest API is used to get the message details and questionnaire. 
  • The Observation API is used to get the member’s adherence profile (PDC scores).
