Reject Code Summary

During the validation process of an immunization file, the following codes and descriptions are generated in the response file to notify the participant that the record was not valid and therefore could not be processed.

Reject Code

Explanation

Field Possibly in Error or Example

1001

There were an incorrect number of elements in the row

 

1003

Unknown segment

There is an extra blank line in the file.

1004

The record does not have a valid Record Type value.

 

1006

Field<#> (<fieldname>) is a required field.

 

1007

Field <#> (<fieldname>) must be equal to or between <min> and <max> length.

 

 

1011

One of the following:

Field <#> (<fieldname>) contains an invalid date format.

Field <#> (<fieldname>) not a valid zip code.

Field <#> (<fieldname>) contains non-numeric data.

Field <#> (<fieldname>) contains non-alphanumeric characters.

Field <#> (<fieldname>) does not match allowable pattern <pattern>.

 

1015

Provider Notification Letter could not be delivered by electronic transmission, fax, or postal mail. The primary care provider address is not deliverable

Primary Care Provider Address 1, Primary Care Provider Address 2, Primary Care Provider City, Primary Care Provider State, Primary Care Provider ZIP Code

1016

Error creating PNL letter.  Note: This is an unusual error, and is not expected to occur.

Examples include unknown brand indicator or intermittent connectivity issues with the Fax vendor.

When processing an immunization file, the generation of PNLs may not be successful for each immunization record in the file. The system will use the following codes and descriptions in the response file to indicate that the record could not be processed due to a problem generating the PNL.

2001

Mandatory PNL field is not present or is invalid.

 

2002

Immunization property not found: <property name>. Where <propertyName> is a property that was included in the template and it is not available or valid.

 

2003

PNL template not found.

 

2004

PNL could not be generated from the template.

 

When processing an immunization file, the delivery of PNL may not be successful for each immunization record in the file. The system will use the following codes and descriptions in the response file to indicate that the record could not be processed due to a problem delivering the PNL.

3001

PNL could not be delivered by electronic transmission, fax, or postal mail. The primary care provider address is not deliverable.

 

7000-7999

Reserved for customer use

Participant may need to use internally assigned error codes for internal processes. These will not be used by Surescripts.

9000

When processing an immunization file, if the system encounters an unexpected problem it will mark the immunization record with code 9000.

An unexpected processing exception occurred.

9001

When processing an immunization file, if the system gets an unexpected error when sending the PNL message it will mark the record with a code 9001 and a description of “Error sending PNL message”.

 

9002

When processing an immunization file, if the system timeouts while attempting to deliver a PNL message it will be mark the record with a code 9002 and a description of “System timed out when send PNL message”.
