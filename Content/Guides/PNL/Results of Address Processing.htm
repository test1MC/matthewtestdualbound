Results of Address Processing

The processed form of each primary care provider address is included in the response file, in addition to the address submitted in the Immunization File. Accompanying the processed address are two additional elements, one indicating whether the address was deemed to be deliverable according to CASS error processing, and another element summarizing the processing outcome.

Address Deliverability

If CASS processing identifies an error that causes an address to be undeliverable, the PCP Address Deliverable element in the response file will contain a value of “N”. Note that a postal address error will not prevent delivery of provider notification via electronic message or secure fax.

Address processing summary information

Information categorizing the outcome of address processing is provided in the PCP Address Processing Summary element. Preliminary values in the summary element include:

  • Valid – Address was modified
  • Valid - Address not modified
  • Unknown street
  • Invalid building number
  • Valid but missing suite
  • Non-USPS
