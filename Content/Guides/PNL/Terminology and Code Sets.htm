Terminology and Code Sets

Vaccine Identification

CVX. The Meaningful Use-specified code set for identifying the type of vaccine administered is the HL7 external code set, CVX.

The CDC’s National Center for Immunization and Respiratory Diseases (NCIRD) maintains the CVX codes, and new codes are added as needed.

See the most current version of this code set here:
http://www.cdc.gov/vaccines/programs/iis/stds/cvx.htm

CPT. Vaccines may additionally be identified using CPT (Current Procedural Terminology) codes maintained by the American Medical Association. A cross-reference between CPT and CVX codes can be found on the CDC website:
http://www2a.cdc.gov/nip/IIS/IISStandards/vaccines.asp?rpt=cpt

CDC/AIRA guide: “CVX code is strongly preferred”. Mapping CPT to CVX is here:
http://www2a.cdc.gov/nip/IIS/IISStandards/vaccines.asp?rpt=cpt
