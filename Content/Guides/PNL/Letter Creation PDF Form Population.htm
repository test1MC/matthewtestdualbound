Letter Creation (PDF Form Population)

A Physician Notification Letter (PNL) will be generated using certain elements contained in the immunization file submitted by the participant.

Letter Format(s)

The participant may supply a logo and static introductory text (including PHI compliance-related statements) to include in notifications sent to primary care providers. This information will be placed into a notification layout which defines the location of:

  • logo/branding
  • static introductory text (content that does not change by patient or immunization), and
  • dynamic elements based on each immunization event:
    • primary care provider name and information
    • patient name and information
    • place of care where the patient was seen, and
    • details of the vaccine administered.

Letter Element Population

All person name elements in the letter will be composed of populated component name fields in the immunization file, omitting middle name if it is not populated, in this order:

First Name + Middle Name (if populated) + Last Name

All addresses in the letter will be presented in two text blocks, as follows:

Text Block 1: Address Line 1 + [, Address Line 2 (if populated)]

Text Block 2: City + “, ” + State Code + ZIP Code

The administered dose element of the letter will be populated with the concatenation of Administered Amount and Administered Units, if populated

Dose: Administered Amount + [Administered Units]

The site and route of administration will be combined into a single text block in the letter, as follows:

Route of administration [+ / Site of administration (if populated)]

Dose number will be populated with Dose Number in Series plus Number of Doses in Series, if it is included in the Immunization File:

Dose Number: Dose Number in Series + [Number of Doses in Series]
