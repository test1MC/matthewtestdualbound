File Naming and Formats

File naming convention is IMMParticipantNameYYYYMMDDXX where XX is a unique sequence for that day. It is strongly suggested to compress the file prior to transmitting it to Surescripts. The current zip protocols that are supported are gzip, bzip2 and zip.

If compression is used, the archive file should contain only a single immunization file. The compressed archive file should have the same file name as the immunization file it contains, with the addition of the file extension of the compression method.

Example: Immunization file name: IMMAcmePharmacy2012112001.txt is then placed in a compressed archived: IMMAcmePharmacy2012112001.txt.gz

The file should then be delivered to Surescripts using FTP and placed in the “to_surescripts” folder.
