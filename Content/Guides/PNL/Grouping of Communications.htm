Grouping of Communications

Where possible, the process will combine immunization notifications in a single method (e.g. one envelope) to reduce the number of messages received by a particular provider.

Patient With Multiple Immunizations

When a patient receives multiple immunizations during a single visit (or during the same day at the same care location), their immunizations will be combined into a single physician notification letter.

The number of immunizations that will fit in a single physician notification letter will depend on the space required by the participant’s standard letter content. Three immunizations can be combined into a single letter.

Immunizations will be grouped into the same letter if all of the information below is the same in the individual immunization records supplied in the Immunization File:

  • Source System ID + Participant’s Patient ID
  • Patient Name
  • Primary care provider’s NPI, name and address
  • Care location at which the immunizations were administered
  • Administration Date
  • Notification Letter Configuration Code (which indicates the letter content and branding to be used).

This grouping occurs during all delivery methods: electronic, fax, and postal mail.

Provider With Multiple Immunizations

When multiple patients of the same provider receive immunizations on the same day, and the provider is reachable only by postal mail, multiple physician notification letters will be delivered in the same envelope.

PNLs will be combined into the same envelope if all of the information below is the same in the individual letters:

  • Primary care provider’s NPI
  • Primary care provider’s name
  • Primary care provider’s address
  • Configuration code.

This grouping occurs only for postal mail delivery, and does not affect electronic or fax delivery.
