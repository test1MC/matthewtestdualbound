Processing Sequence

Overall file validation

  • If errors exist in over 10% of the individual immunization file records, or if a partial file was received, then the file will be rejected with a file-level error message.

Individual data validation

  • The system validates field formats, lengths, and valid values, and confirms mandatory elements are present and conditionality requirements are met.

Physician Notification Letter Delivery

The Physician Notification Letters for Immunizations product uses Surescripts Guaranteed Delivery capability to determine how letters will be routed. Guaranteed Delivery operates by matching data in the PNL record to data in the Surescripts directory and makes routing decisions through an algorithm. The Guaranteed Delivery algorithm sequences as follows:

  • The system determines whether the primary care provider is accessible electronically. If so, it initiates an electronic message containing a PDF of the physician notification letter.

    If delivery is not successful, the system attempts fax delivery.
  • If the participant provides a physician’s fax number in the immunization record, the system will attempt to deliver the physician notification letter to the physician’s fax number.
  • If the participant does not provide a fax number or fax delivery fails, then the system will attempt delivery using postal mail.
  • Surescripts will use the mailing address for the primary care provider provided in the file to mail the physician notification letter. The system will validate and format the address according to the USPS CASS process. If the address is determined to be deliverable, then the system prints the physician notification letter(s) and they are sent for delivery by postal mail.

    The physician letter will be delivered through only one method.

Response file creation process

  • Once all immunizations have been processed, the system creates the Immunization Delivery Report described in a later section.
  • The response file contains the processing outcome of each record, including:
    • method by which delivery of the physician notification letter was initiated (electronic, fax or postal mail)
    • notice when delivery was not attempted due to a physician postal address that was not USPS-deliverable
    • other processing errors encountered
    • the fax number used (when fax is delivery method)
    • scrubbed primary care provider address (Surescripts will validate and format primary care provider postal addresses according to the USPS Coding Accuracy Support System (CASS process), and
    • a field indicating that one or more address elements were modified by CASS processing (e.g., ZIP Code added, street directional added).
