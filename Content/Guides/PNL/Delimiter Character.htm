Delimiter Character

The following table contains delimiters used in inbound and outbound files.

The participant must not use any of the following delimiters within data fields. If included within the file record, the record will be flagged as an error and will not be loaded.

Character

Description

ASCII Hex Value

*

Asterisk

0x2a

|

Pipe

0x7c

\

Backslash

:

Colon

0x3a

~

Tilde

0x7e

LF

Line Feed

0x0a

CR
