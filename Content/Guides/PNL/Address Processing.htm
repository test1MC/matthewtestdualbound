Address Processing

During processing of the Immunization File, Surescripts “scrubs” each primary care provider (PCP) address present in the file in order to:

  • correct address elements, such as ZIP Code, where possible
  • standardize the formatting of the address
  • determine deliverability of postal mail to the address, and
  • remove non-address content from the second address line when present.

Surescripts performs address correction, formatting, and deliverability validation according to the USPS CASS process.
