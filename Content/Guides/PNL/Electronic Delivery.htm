Electronic Delivery

Primary care provider notification will be delivered electronically via the Surescripts Clinical Interoperability network to those who subscribe to the network.

Elements Required for Electronic Delivery

Electronic recipients are located using primary care provider identifiers, name and address information contained in the Immunization File. At a minimum, the following elements must be present to enable electronic delivery:

  • Primary Care Provider NPI
  • Primary Care Provider name elements
  • Primary Care Provider Address Line 1
  • Primary Care Provider ZIP Code
