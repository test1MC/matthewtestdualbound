RES-040

FREE TEXT

<Note>

an..70

All other messages.

XML tag <note> is used for Approved responses and XML tag <DenialReason> is used for Denied responses, including DeniedNewPrescriptionToFollow.

See reason code tables below for use in this field.

<Note> OR

<DenialReason>

For Cancel Rx Response where the prescription has been transferred out to another pharmacy use the denied code in RES-010 along with the text "Rx was transferred to" in this field. Additional information such as the store name and number can be included after the text defined above.

RxFill Reason Codes for Dispensed or Partially Dispensed

These codes and descriptions are for use in the Free Text - RES-040 <Note>.

Description
