Message Linkage

In order to link messages together four key fields are defined below. This linkage allows a software system to tie one message to another. For example, on a Refill Renewal Request the prescribing system needs to find the original NewRx to which the pharmacy is referring. This linkage should be done in an automated manner.

UIB-030-01 Transaction Control Reference/<MessageID> – The unique number assigned to each message.

UIB-030-02 Initiator Reference ID/<RelatesToMessageID> –The UIB-030-01 <MessageID> of the message being responded to.

UIH-020      Message Reference Number/<RxRefereneNumber> – The prescription number assigned by pharmacy system.

UIH-030-1   Initiator Control Reference/<PrescriberOrderNumber> – The prescription number created by the prescribing system. For diagram below this is called PON.
