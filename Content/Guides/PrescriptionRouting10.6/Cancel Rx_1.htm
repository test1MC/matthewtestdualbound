Cancel Rx

In this example, a Cancel Rx Request is sent from the prescriber (prescriber system) to the pharmacy through Surescripts.

In the EDIFACT examples, line breaks are used at the end of the segments for display purposes – live messages should not contain line breaks.

CANRX (EDIFACT)

UNA:+./*'

UIB+UNOA:0++666333:991+++1234567890123:D::12345+1234563:P::678+20100125:081322'

UIH+SCRIPT:010:006:CANRX+1515+++20100125:081522'

PVD+PC+1234567890:HPI*3334444:0B+++JONSON:TIM++++6518659191:TE'

PVD+P2+1234567899:HPI*1234563:D3+++++MCMOHRS CORNER PHARMACY++6518952323:TE'

PTT+1+19521001+WHITESIDE:KARA+F+YX-5-VA-20011246:2U+23230 SEAPORT:AKRON:OH:44306'

DRU+P:LIPITOR 10MG TABLET:00071015523:ND+:30:38:AC:C48542+:TAKE 1 TABLET BY MOUTH EVERYDAY+85:20100125:102*ZDS:30:804+1+R:0'

UIT+1515+6'

UIZ++1'

Segment

Value

Note

UIB-010

UNOA:0

UNOA is the Syntax identifier and 0 is the syntax version number.

UIB-030

666333:991

666333 is the Message ID; 991 is the Relates To Message ID, which is the Message ID from the NewRx being canceled.

UIB-060

:1234567890123:D::
12345

1234567890123 is the identification number (SPI) of the sender (prescriber); D is the code qualifier for prescriber; 12345 is the Surescripts-populated Account ID for the sender (prescriber).

UIB-070

1234563:P::678

1234563 is the identification number (NCPDP ID) for the receiver (pharmacy); P is the code qualifier for pharmacy; 678 is the Surescripts-populated Account ID for the receiver (pharmacy).

UIB-080

20100125:081322

20100125 is the date the message was sent; 081322 is the time the message was sent (08:13:22 A M.).

UIH-010

SCRIPT:010:006: CANRX

SCRIPT is the message type, 010 is the version, and 006 is the release to be used in decoding this message. All messages in this set use this value. CANRX is the message function for Cancel Prescription.

UIH-020

1515

1515 is the prescription number assigned by the pharmacy.

UIH-050

20100125:081322

20100125 is the date the message was sent (01/25/2010) 081322 is the time the message was sent (08:13:22 A.M.).

PVD-010

PC

PC identifies this PVD segment as information about a prescriber.

PVD-020

1234567890:HPI*3334444:0B

1234567890 is the prescriber's reference number; HPI is the qualifier for NPI; 3334444 is the reference number of the prescriber; 0B is the qualifier for State License Number.

PVD-050

JONSON:TIM

Johnson is the last name of the prescriber; Tim is the first name.

PVD-090

6518659191:TE

6518659191 is the Communication Number for the prescriber, (651-865-9191); TE is the qualifier for telephone.

PVD-010

P2

P2 identifies this PVD segment as referring to pharmacy.

PVD-020

1234567899:HPI*1234563:D3

1234567899 is the pharmacy's reference number; HPI is the qualifier for NPI; 1234563 is the reference number for the pharmacy and D3 is the qualifier for NCPDP ID.

PVD-070

MCMOHRS CORNER PHARMACY

McMohr's Corner Pharmacy is the name of pharmacy.

PVD-090

6518952323:TE

6518952323 is the Communication Number for the pharmacy (651-895-2323); TE is the qualifier for telephone.

PTT-010

1

1 is the patient relationship code of member.

PTT-020

19521001

19521001 is the patient's date of birth 10/01/1952.

PTT-030

WHITESIDE:KARA

WHITESIDE is the last name of the patient; KARI is the first name.

PTT-040

F

F indicates the patient is female.

PTT-050

YX-5-VA-20011246:2U

YX-5-VA-20011246 is the PBM Unique Member ID; 2U is the qualifier for payer ID.

PTT-060

23230 SEAPORT:AKRON:OH: 44306

23230 SEAPORT is Address Line 1 for the patient; AKRON is the city; OH is the state and 44306 is the zip code.

DRU-010

P:LIPITOR 10MG

P is the medication prescribed; LIPITOR 10MG is the name of the prescribed drug; 00071015523 is the drug number and ND is the code list indicator for NDC11.

DRU-020

:30:38: AC:C48542

30 is the number of tablets; 38 is the code value for Original Qty; AC is the Source for NCI Potency Units and C48542 is the code for Tablet dosing unit.

DRU-030

TAKE 1 TABLET BY MOUTH EVERYDAY

TAKE 1 TABLET BY MOUTH EVERYDAY indicates the dosage instructions or the SIG.

DRU-040

85:20100125:102*
ZDS:30:804

85 is the qualifier for Date Written; 20100125 indicates the date and 102 is the format qualifier for CCYYMMDD for 01/25/2010.

Second loop for another date ZDS is the code for Days Supply; 30 is the number of days supply; 804 is the qualifier for Quantity of Days.

DRU-050

1

1 indicates no substitutions are allowed.

DRU-060

R:0

R indicates Number of refills; 1 is the quantity for the number of refills.

UIT-010

1515

1515 is the message reference number assigned by the pharmacy. This must match the value in UIH-020.

UIT-020

6

7 is the number of segments in the message between and including the UIH and UIT segments.

UIZ-020

1

1 is the Interchange Control Count – the number of messages in this interchange.
