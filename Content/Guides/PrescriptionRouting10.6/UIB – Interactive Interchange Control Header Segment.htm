UIB-060

INTERCHANGE SENDER

 

M

UIB-060-01

Sender ID – Level One

<From>

an..35

This field contains the identification number of the sender. This field is used in conjunction with UIB-060-02 Level one code qualifier, which qualifies which ID was used.

It will contain the NCPDP Provider ID of the pharmacy or the SPI (Surescripts Provider ID) of the prescriber.

M

UIB-060-02

Level One ID Code Qualifier

<From Qualifier>

an..4

Values:

D = Prescriber ID (Used for SPI)

P = Pharmacy NCPDP ID

ZZZ = Surescripts network customer name, should only be used in rare cases when a partner gateway needs to send an ERROR or VERIFY message and an NCPDPID or SPI is not available (i.e. when the message causing the error could not be parsed).

UIB-080-02

Event Time

<SentTime>

YYYY-MM-DDTHH:MM:SS.FZ

YYYY-MM-DDTHH:MM:SS.F+00:00

YYYY-MM-DDTHH:MM:SS.F-00:00

n8

Seconds must be included, but fractional seconds is not required.

The UTC time zone may be specified as Z, + 00:00, or - 00:00. Refer to xsd:dateTime for more information.

Note: Although the type is defined as numeric, the comma is included in this field as a placeholder.
