UNA – Service String Advice Segment

The UNA EDIFACT segment is a fixed-length segment. This segment determines the delimiters for the message which follows. The customer should use the values of each separator as defined below, which are in decimal representation with the Hex value in parenthesis. Surescripts recommends using unprintable delimiters, as printable delimiters have caused issues in the past. Please refer to EDIFACT Dynamic Delimiters on page 1.
