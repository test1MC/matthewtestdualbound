UIH-020

MESSAGE REFERENCE NUMBER

<RxReferenceNumber>

an..35

This is the prescription number assigned by the pharmacy.

M = Refill Renewal Request, Refill Renewal Response

C = All other messages

Prescriber applications should be able to accept duplicate Rx Reference Numbers, as it is possible for a pharmacy to send duplicate Refill Renewal Request messages if a Refill Renewal Response is not received in a timely manner.
