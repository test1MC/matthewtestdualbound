Note

UIB

991

Transaction Control Reference that is the prescriber system's trace number.

UIB

1234567890123:D

1234567890123 is the SPI for the prescriber; D is the code qualifier for prescriber.

UIB

NCPDP:P

NCPDP is the NCPDP ID for the pharmacy; P is the code qualifier for pharmacy.

UIB

20100125:081322

Date and time message was sent 01/25/2010 08:13:22 A.M.

UIH

SCRIPT:010:006

This defines the message types and version to be used in decoding this message. All messages in this set use this value.
