Adherence Profile Summary

This will return the patient’s lowest PDC value, disease class.

Adherence Profile Summary Template Example

[GET] {{host}}/mmpoc/api/v0/adherence-profiles/summary?memberId={Patient PBM Unique ID]&payerId={Payer Id}

Request JSON Fields

M/C - Mandatory or Conditional

JSON Filed

Data Type

M/C

Description

memberId

an 60

M

Patient PBM unique Id.

payerId

an 15

M

PBM/payer’s Participant Id.

Adherence Profile Summary Request Example

[GET] {{host}}/mmpoc/api/v0/adherence-profiles/summary?memberId=ZZQ-88-10011-249&payerId=T00000000001000

Response Field

Description

pdcLastUpdated

Date the PDC ratio was last updated by the PBM/payer.

pdcWorstStatus

Indicates the status of the lowest PDC score for the patient based on the ratio value (healthy, moderate or low)  ** more info below.

pdcValue

Integer representing the patient's lowest PDC value.

pdcValueUnits

Units associated with the PDC value.

pdcDiseaseClass

The disease class name (possible values are: cholesterol, hypertension, diabetes).
