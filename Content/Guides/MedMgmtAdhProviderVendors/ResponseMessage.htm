Response Message

Responding to a medication management message is strongly encouraged, but is optional. The provider will be given the ability to select a response from a drop down list or provide a free-form response. This feedback will be stored in a Surescripts repository and can be queried for by PBM/payers at any given time.

Data elements provided to the PBM/payers upon query are the following:

Data Element Name

Description

PBM Message Id

PBM/payer’s unique message Id associated with the medication management message.

Message Type

Message type of HRM, Missing Medication or Medication Adherence.

Patient Id, Patient Name, Patient DOB

Patient information supplied by the PBM/payer.

Received Date

Date the message was received and the workflow process was initiated.

Message Viewed Date

Date the workflow task associated with the PBM/payer message was accepted by a user.

Message Not Viewed Date

If a message is not viewed and response was not submitted on the date the task expires.

Feedback Complete Date

The date that a response was submitted.

Workflow Process Id

The workflow process Id the PBM/payer will need when acknowledging the receipt of disposition data for a given message.

Workflow Task Id

The workflow task Id the PBM/payer will need when acknowledging the receipt of disposition data for a given message.

Feedback response date, question and answer

The response submitted by the user and the user Id from the authentication token.

Feedback response optional comment

If a comment was submitted by the user and the user Id from the authentication token.
