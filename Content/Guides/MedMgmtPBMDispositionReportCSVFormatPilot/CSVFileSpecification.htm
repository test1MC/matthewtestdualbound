MessageId

Unique message ID

Unique message Id from the PBM/payer system which was included in the message sent to Surescripts by the PBM/payer. 
This message Id is used by the PBM to link the disposition/results to the original message/opportunity in the PBM/payer system.

Example: 5bd3fa644927901e763880ec6891
