Message Process Flow

  1. PBM/payer sends HRM, Missing Medication, Medication Adherence and PDC messages to Surescripts.
  2. Surescripts receives and validates the message.
  3. Provider vendor end user views the message (viewed).
  4. Provider vendor end user optionally submits a feedback response to the HRM, Missing Medication or Medication Adherence message (response).
  5. On an agreed upon cadence, the Disposition Report in CSV format is generated.
  6. The PBM/payer pulls the CSV report from the Surescripts SFTP site.
