About This Guide

Surescripts created this document to describe the Disposition Report CSV (comma-separated values) format and file exchange using the SFTP process for pharmacy benefit managers (PBMs) in implementing the Medication Management for Adherence. This process will be used for the duration of the PBM/payer vendor pilot.
