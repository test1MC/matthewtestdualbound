Organization Reports

Organization Transaction Report

This text file lists each transaction associated with organizations of the account. Information is at a high level and includes NCPDPID, SPI, Message Type, Date and Time of the Transaction, and the customer’s Message ID.

Organization Monthly (Activity Report)

This PDF file aggregates activity for account transactions.

NEWRX2REFREQ Report

This report is designed to identify for organizations the ratio of incoming NewRx to the RefillRenewal Requests generated to a specific provider. Inclusion on this report may identify issues with generating electronic refill requests to a provider.

Organization Monthly Analysis Report

This report provides a three month trend review of transaction and network-related analysis.
