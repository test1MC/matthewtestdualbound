Case Management Link

Click on Case Management in the left menu. The Active window will display a New Case link and a date range to search for open or closed cases through Admin Console.

To create a new case, click on the New Case button and the support case template displays.

Patient PHI (Protected Healthcare Information) is not permitted within the case details in the Self-Service Portal or the Admin Console.

Complete the appropriate fields to highlight information to support the case. As with the Self-Service Portal, some fields have a red asterisk (*) and must be included to successfully submit the case. Where provided, be sure to indicate the appropriate case type and priority you wish to assign to the case.

Case Management

Managing cases from the Admin Console is limited to only those cases created within the Admin Console interface. Cases created in the account-holder’s Self-Service Portal interface will not be visible in the Admin Console.

To view all cases for an account, you must utilize the Self-Service Portal interface. To view cases created in the Admin Console, click on Case Management in the left menu and set the date range under the New Case link. By default, open cases will display. Click the Closed Cases Only check-box to display cases created in the Admin Console which have since been resolved and closed.

To access case details, click on the Case Number link. The case details will appear including any attachments or comments that have been appended to the case. The user can add case comments and attachments as needed.

!! Important !! If attaching a document that contains PHI, the user must first password protect and encrypt the document to ensure the patient information is not exposed. Once uploaded the user should contact Surescripts Customer Care to provide the associated password.
