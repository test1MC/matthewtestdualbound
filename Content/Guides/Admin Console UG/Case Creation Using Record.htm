Case Creation Using Record Details

Within the provider and organization record details, click the Open a Support Case link.

To create a case from a specific message detail, click the Create Support Case link. The resulting window (shown below) will be similar to the new case window as described in Case Management Link. Specific information on the provider, organization, or message details will pre-populate the form. All other steps in completing and transmitting the case apply.

To manage a case, refer to Case Management.
