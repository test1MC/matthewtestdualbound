About This Guide

The Surescripts Master Patient Index (MPI) File Load Supplement is a supplement to Surescripts National Record Locator Service (NRLS). This supplement was created to detail the MPI file load, which is used to load the participant’s patients into the Surescripts MPI.

The audience for this document is the participant development team responsible for building the software necessary to interact with Surescripts for the implementation of the MPI File Load in their application system.
