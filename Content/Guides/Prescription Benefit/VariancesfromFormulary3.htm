Variances from Formulary 3.0

Field

Field Description

Variance

Reason for Variance

Product/Service ID Qualifier

Drug ID qualifier

Only NDC is supported.

To date NDC is the only universally accepted product identifier.

Product/Service ID Source

Drug ID

Surescripts requires N 11/11.

Mandatory because Class ID is no longer used.

NDCs are numeric and 11 digits in length.

Since no class ID this is the only way to identify the product.

List Action

Tells the receiver that this is a Full list replacement (Or Add) or a delete list. 

The update option is not currently supported

Senders have only implemented full file replacement.

Change Identifier

Change

Only the Add option is accepted.

Senders have only implemented add option.

Message Short

Text message presented to prescriber.

Mandatory for this implementation.

Standard is in error indicating this is optional. If sending a text message record you need the message.

Product/Service ID Qualifier – Step Drug

Drug ID qualifier.

Required.

Since no class ID this is the only way to identify the product.

Drug Reference Qualifier

Code value that identifies the source and type for the Drug Reference Number.

Only support a limited set of code values.

Surescripts will not be supporting other drug databases.
