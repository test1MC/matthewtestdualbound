List Action

Tells the receiver that this is a Full list replacement (Or Add) or a delete list. 

 

AN 1/1

M

F = Full List Replacement  (If exists, replace, if not, add)

U = Update List

D = Delete List

Note: The detail records list the drugs within the formulary. Detail records may not exist if the non-listed formulary statuses from the header record are used exclusively to convey the drugs’ formulary statuses. If the list action in the formulary status header is a “D” (Delete), the detail records should be ignored (if present) by the retriever.

Variance from Formulary and Benefit 3.0: The update option is not currently supported.
