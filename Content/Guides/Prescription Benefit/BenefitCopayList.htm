Product/Service ID Qualifier

Drug ID qualifier

AN 2/2

M

01 = Universal Product Code (UPC)

02 = Health Related Item (HRI)

03 = National Drug Code (NDC) - This can be the representative NDC Number.

09 = Health Care Financing Administration Common Procedural Coding System (HCPCS)

28 = Universal Product Number (UPN)

Variance from Formulary and Benefit 3.0: Only NDC is supported by Surescripts.
