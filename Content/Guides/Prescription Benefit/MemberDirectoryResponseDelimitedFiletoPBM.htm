Member Directory Response Delimited File to PBM/payer

The file name reflects the file name from the PBM/payer with a .rsp file extension.

Example: NewPatient_TestingPBMC_MPI.1450188243095.rsp

If the source file had an extension, the extension remains and the .rsp is added after it. For example: NewPatient_TestingPBMC_MPI.1450188243095.txt.rsp
