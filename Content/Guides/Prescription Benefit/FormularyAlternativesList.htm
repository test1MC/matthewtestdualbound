Record Type

Identifies record type.

AN 3/3

M

Alternative ID

The identification number for this alternative list.

AN 1/10

M

The ID for the alternative list that may have been returned in the Eligibility 271 message (located in loop 2110C1, REF segment, qualifier = ‘ALS’).

Must be unique across all lists of this type.

Valid characters are (A-Z, a-z, Numeral 0-9, period “.”, and a dash “-“)

List Action

Tells the receiver that this is a Full list replacement (Or Add) or a delete list. 

The update option is not currently supported.

AN 1/1

M

F = Full List Replacement  (If exists, replace, if not, add)

U = Update File

D = Delete List

Note: If the Header Transaction Action is a Delete, the detail records should be ignored (if present) by the Retriever.

M
