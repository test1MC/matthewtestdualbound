XML NextQuestionID Clarification for PA Question Type

Per the NCPDP SCRIPT Standard Format Implementation Guide (v2013101), the NextQuestionID identifies the question the end user should answer next when it is dependent on the answer to the question. See Document References for more detail.

This field is required for the following QuestionTypes:

  • Numeric/NextQuestionCondition/SingleComparison
  • Numeric/NextQuestionCondition/RangeComparison

When QuestionType is Select and DefaultNextQuestionID is not provided, NextQuestionID is a required field. This field is only required where QuestionType is Select and DefaultNextQuestionID is not provided.

If the next question in a question set is not dependent on a specific response, PBM/payers should utilize DefaultNextQuestionID at the question level as opposed to including the same NextQuestionID for each response option.
