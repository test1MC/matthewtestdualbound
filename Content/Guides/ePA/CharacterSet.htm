Character Set

The character set contains ASCII values 32 – 126, which include:

Letters, upper and lower caseA to Z, a to z

NumeralsØ to 9

Symbols! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

Unprintable characters, such as control characters, are not used within the field sets.

Most symbols should be ‘safe’ within most alphanumeric fields; however, XML validation has issue with the following characters and they must be escaped. The most common escape encodings are:

ASCII Characters

XML Escape

String lengths are calculated based on the un-escaped value. For example, “>98 degrees” should be encoded as “&gt;98 degrees” but would only be counted as a string length of 11, not 14.
