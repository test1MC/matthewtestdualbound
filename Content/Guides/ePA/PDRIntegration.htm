PDR Integration

Surescripts has partnered with Physicians’ Desk Reference (PDR) to provide prior authorization coverage for nearly 100 percent of patients on the Surescripts network. This coverage comes from Surescripts’ direct electronic connections to PBM/payers using CompletEPA® and PDR’s extensive forms library. For provider vendors that choose to implement with PDR, Surescripts will be able to return a link to a dynamic PDF form in cases where Surescripts cannot provide an electronic route to the prior authorization. See Electronic Prior Authorization Message Flow for a detailed workflow outlining the process for invoking and returning a PDR form.

Provider vendors need to contract with PDR separately in order for Surescripts to return the URL to the PDR form. In the PANote field, Surescripts will send the PDR Url - [URL]. For example, PDR Url - http://www.google.com.

Due to security restrictions in place at some organizations, prescriber vendors and health systems should ensure that PDR URLs are added to a list of allowed (or whitelisted) sites. For details on specific URLs to be added, please work with your Surescripts Integration Project Manager.
