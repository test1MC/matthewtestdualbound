About This Guide

This guide is a supplement to the Directories 4.4, 4.5 and 4.6 implementation guides and is intended to provide information to provider and pharmacy vendors considering adding the support of Learning Directory. For details on Directories, please refer to the appropriate Surescripts Directories implementation guide.

The Learning Directory simplifies the registration and management process for providers that practice at multiple locations, addressing the significant misalignment between provider information received in NewRx messages and provider information registered in the Surescripts directory. Learning Directory does not replace management of the primary provider location and service levels.

Pharmacies benefit through improved and additional matching opportunities when initiating a Refill Renewal Request or an RxChange request. Provider vendors benefit from a streamlined, automated registration process for these additional care delivery locations.
