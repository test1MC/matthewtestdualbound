Prescription Routing General Best Practices

The following general best practices apply to all prescription routing message types.

  1. If customers implement any type of prescription message store and forward methodology, e.g. hub or central location, the storage/hold time period should be no more than 1-2 minutes.
  2. To maximize the use of electronic transmission, customers should default to electronically send within their application workflow when the receiver is electronically enabled.
  3. Application should not duplicate messages; i.e. electronically delivered as well as faxed.
  4. Applications should have the ability to send ICD-9 or ICD-10 information in all message types. This is because ICD-9/ICD-10 coding is required on some electronic claims. If the information is not included, it could result in a manual process between the provider and pharmacy.
