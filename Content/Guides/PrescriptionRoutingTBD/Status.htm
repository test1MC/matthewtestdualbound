Status

The Status message is used to indicate the message was accepted of a message back to the sender. A Status message is used in response to a NewRx, Refill Renewal Request, Rx Change Request, Rx Change Response, Cancel Rx Request, Cancel Rx Response, RxFill, Resupply, Census or Refill Renewal Response to indicate acceptance and responsibility for a request.

The Status message is an NCPDP message that indicates the acceptance of the request. Surescripts is utilizing the Status message for all messages.

This message is used to indicate the message was accepted for processing.

A Status message can only be used in the HTTP response to a message synchronously.

A Status code of 010 indicates the process is complete. Subsequent acknowledgement messages should not be expected.
