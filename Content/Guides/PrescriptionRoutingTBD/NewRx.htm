NewRx

In this example, a new prescription is sent from the prescriber system to the pharmacy through Surescripts. This is for a non-controlled substance.

Segment

Value

Note

UIB-010

UNOA:0

UNOA is the Syntax identifier; 0 is the syntax version number.

UIB-030

991

991 is the Transaction Control Reference.

UIB-060

1234567890123:
D::12345

1234567890123 is the identification number (SPI) for the sender (prescriber); D is the code qualifier for prescriber; 12345 is the Surescripts-populated Account ID for the sender (prescriber).

UIB-070

1234563:P::678

1234563 is the identification number (NCPDP ID) for the recipient (pharmacy); P is the code qualifier for pharmacy; 678 is the Surescripts-populated Account ID for the recipient (pharmacy).

UIB-080

20100125:081322

20100125 is the date the message was sent (01/25/2010); 081322 is the time the message was sent (08:13:22 A.M.).

UIH-010

SCRIPT:010:006:
NEWRX

SCRIPT is the message type, 010 is the version, and 006 is the release to be used in decoding this message. All messages in this set use this value. NEWRX is the message function for New Prescription.

UIH-030

PON123

PON123 is the prescription number assigned by the prescribing system.

UIH-050

20100125:081522

20100125 is the Date the message was sent (01/25/2010); 081522 is the time the message was sent (8:15:22 A.M.).

PVD-010

PC

Identifies this PVD segment. PC identifies this PVD segment as information about a prescriber.

PVD-020

1234567890:HPI*
3334444:0B

1234567890 is the prescriber's reference number; HPI is the qualifier for NPI; 3334444 is the reference number of the prescriber; 0B is the qualifer for State License Number.

PVD-050

JONSON:TIM

Johnson is the last name of the prescriber; Tim is the first name.

PVD-080

1234 FOURTH ST:
ST PAUL:MN:55101

1234 FOURTH ST is Address Line 1 for the prescriber; St Paul is the city; MN is the state; 55101 is the prescriber address zip code.

PVD-090

6518659191:TE

6518659191 is the communication number for the prescriber (651-865-9191); TE is the qualifier for telephone.

PVD-010

P2

P2 identifies this PVD segment as information about a pharmacy.

PVD-090

1234567899:HPI*
1234563:D3

1234567899 is the pharmacy's reference number; HPI is the qualifier for NPI; 1234563 is the reference number of the pharmacy; D3 is the qualifier for NCPDP ID.

PVD-070

MCMOHRS CORNER
PHARMACY

McMohr's Corner Pharmacy is the name of the pharmacy.

PVD-080

123 THIRD St:
St PAUL:MN:55123

123 THIRD St is Address Line 1 for the pharmacy; St PAUL is the city; MN is the state and 55123 is the zip code.

PVD-090

6518952323:TE

6518952323 is the Communication number for the pharmacy; TE is the qualifier for telephone.

PTT-010

1

1 is the patient relationship code of member.

PTT-020

19521001

19521001 is the patient's date of birth 10/01/1952.

PTT-030

WHITESIDE:KARA

WHITESIDE is the last name of the patient; "KARI" is the first name.

PTT-040

F

F indicates the patient is female.

PTT-050

YX-5-VA-20011246:2U

YX-5-VA-20011246 is the PBM Unique Member ID; 2U is the qualifier for payer ID.

PTT-060

23230 SEAPORT:AKRON:
OH:44306

23230 SEAPORT is Address Line 1 for the patient; AKRON is the city; OH is the state and 44306 is the zip code.

DRU-010

P: LIPITOR 10MG TABLET:
00071015523:ND

P indicates prescribed; LIPITOR 10MG Tablet is the prescribed drug; 00071015523 is the drug number; ND is the qualifier for NDC11.

DRU-020

:30:38: AC:C48542

30 is the quantity of tablets; 38 indicates the original quantity prescribed; AC indicates the Potency Unit; C48542 is the code for Tablet dosing unit.

DRU-030

TAKE 1 TABLET BY
MOUTH EVERYDAY

TAKE 1 TABLET BY MOUTH EVERYDAY indicates the dosage instructions or SIG.

DRU-040

85:20100125: 102*ZDS:
30:804

85 is the qualifier for Date Written; 20100125 indicates the date and 102 is the format qualifier for the format of CCYYMMDD for 01/25/2010.

Second loop for another date. ZDS is the code for Days Supply; 30 is the number of days supply; and 804 is the qualifier for Quantity of Days.

DRU-050

1

1 indicates no substitutions are allowed.

DRU-060

R:0

R is the qualifier for Number of Refills; 0 indicates the quantity of refills allowed is zero.

COO-010

031430643:ZZ*610415
:BO*T0000000002163
3:2U

031430643 is the ISA Control Number; ZZ is the qualifier for mutually defined; 610415 is the "BIN number; BO is the qualifier for BIN number; T00000000021633 is the payer ID; 2U is the qualifier of payer ID.

COO-020

PBMF

PBMF is the name of Payer.

COO-040

Member ID

Member ID is the Cardholder ID.

COO-050

WHITESIDE:KARA

WHITESIDE is the Last name of the cardholder; KARA is the first name.

COO-060

TTC1

TTC1 is the Group ID.

UIT-020

7

7 is the number of segments in the message between and including the UIH and UIT segments.

UIZ-020

1

This is the Interchange Control Count – the number of messages in this interchange. For this example it is one.
