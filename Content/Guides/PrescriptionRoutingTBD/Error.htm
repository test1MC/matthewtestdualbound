Error

This message indicates that an error has occurred. An error can be generated when there is a communication problem or when an error occurs during the processing of the message.

An Error message can either be free-standing (asynchronous) or as an HTTP response.

An Error indicates the process is complete. Subsequent acknowledgment messages should not be expected.

A synchronous error is sent in response to a message before breaking the communication connection. An asynchronous error message (also called Free Standing Error message) is sent in response to a message any time after the Status 000 message is sent.

An Error message may not be used to respond to an Error message.
