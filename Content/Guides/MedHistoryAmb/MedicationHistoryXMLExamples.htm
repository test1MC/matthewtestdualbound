Medication History XML Examples

In this example, the provider vendor is requesting historical medication information for a patient. The provider vendor has already determined the PBM Unique ID through the eligibility message. This unique number will be passed within the message.

Medication History Request Claim Data (PBM/Payer)

<?xml version="1.0" encoding="UTF-8"?>

<Message xmlns="http://www.ncpdp.org/schema/SCRIPT" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" version="010" release="006" xsi:schemaLocation="http://www.ncpdp.org/schema/SCRIPT SS_SCRIPT_XML_10_6MU.xsd">

  <Header>

    <To Qualifier="ZZZ">PBM123</To>

    <From Qualifier="ZZZ">1234567890123</From>

    <MessageID>991</MessageID>

    <RelatesToMessageID>987</RelatesToMessageID>

    <SentTime>2017-10-01T08:13:22</SentTime>

    <Security>

      <Sender>

        <SecondaryIdentification>123456987</SecondaryIdentification>

      </Sender>

      <Receiver>

        <SecondaryIdentification>RXHUB</SecondaryIdentification>

      </Receiver>

    </Security>

  </Header>

  <Body>

    <RxHistoryRequest>

      <Prescriber>

        <Identification>

          <NPI>1234567890</NPI>

        </Identification>

        <Name>

          <LastName>JONSON</LastName>

          <FirstName>TIM</FirstName>

        </Name>

        <CommunicationNumbers>

          <Communication>

            <Number>6515559191</Number>

            <Qualifier>TE</Qualifier>

          </Communication>

        </CommunicationNumbers>

      </Prescriber>

      <Patient>

        <PatientRelationship>1</PatientRelationship>

        <Name>

          <LastName>JAMES</LastName>

          <FirstName>TINA</FirstName>

        </Name>

        <Gender>F</Gender>

        <DateOfBirth>

          <Date>1974-06-05</Date>

        </DateOfBirth>

      </Patient>

      <BenefitsCoordination>

        <PayerIdentification>

          <PayerID>PBM123</PayerID>

        </PayerIdentification>

        <PayerName>PBMCOMPANY</PayerName>

        <EffectiveDate>

          <Date>2017-01-01</Date>

        </EffectiveDate>

        <ExpirationDate>

          <Date>2017-08-01</Date>

        </ExpirationDate>

        <Consent>Y</Consent>

        <PBMMemberID>PBM11356</PBMMemberID>

      </BenefitsCoordination>

    </RxHistoryRequest>

  </Body>

</Message>
