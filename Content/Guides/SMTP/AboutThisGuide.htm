About This Guide

This Surescripts SMTP Edge Protocol(SMTP) XDR Implementation Guide was created to assist customers in developing and transferring messages needed to provide secure clinical information exchange between providers. The audience for this document includes any customer responsible for developing a system interface to the Surescripts network using SMTPXDR. This document assumes that the customer is already familiar with SMTPXDR and has a working SMTPXDR implementation.
