Introduction

The Certification & Testing Suite is a testing component of the Admin Console. This staging environment tool is available to participants prior to their certification for production. If you have questions concerning this guide or the Certification & Testing Suite, please direct them to your Surescripts Account Manager or email the Surescripts Customer Care Team at support@surescripts.com.

About This Guide

This guide was created to assist pharmacy and vendor participants in their use of the “Suite” permitting them to trigger messages that each would expect to receive in a production environment.

This guide addresses the Suite tools and functionality, impersonation, selection of response and non-response messages, substitution of real world patient data, randomization of data blocks, the setup and execution of test cases, and reporting.

Components

Upon logging into the Admin Console, you can access the Certification Suite through the left-hand menu. The key elements of this suite include:

  • Impersonation
  • Administration
    • Test Case Administration
    • Reporting
    • Profile Management
    • Medications Management

General Interface Standards

The following interface standards are used throughout the Certification and Testing Suite screens.

  • “Breadcrumbs” at the top of each screen tell you where you are within the Certification & Testing Suite application. For example, the user below is on the Main screen of the Certification Suite Home page.

  • Search field qualifiers: any combination of fields may be used to conduct searches.

Not all search filters have search field qualifiers.

 
Qualifier Description
E Exactly matched. The field must contain an exact match of information.
P Partially-matched. The search will match the characters typed in the field. For example, "more" will yield results that start with "more" such as: "moreland" and "morefield"; it will not yield a result such as "Kenmore".
1 This superscript is reserved for SPI numbers and must include either the 10-digit SPI root or the full 13-digit SPI number.
