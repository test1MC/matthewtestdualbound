Messages Overview

Message Description(s)

PMBC Check Request

The PMBC Request is a request message from a prescriber to the PBM/payer to obtain real-time, patient specific benefit information based on the selected pharmacy, medication, and days supply combination.

PMBC Response

The PMBC Response is a response message from the PBM/payer to the prescriber with the status of the PMBC Request and the patient’s coverage information.

Error Response Message

ERROR (Required)

This NCPDP SCRIPT message indicates that an error has occurred and the PMBC Request has been terminated. An error can be generated when there is a communication problem or when the message had an error (e.g. a formatting problem).
