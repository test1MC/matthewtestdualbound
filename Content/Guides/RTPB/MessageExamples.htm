PMBC Request

In this example, the initiator of the PMBC Request is a Prescriber using vendor EMRCOMPANY. The prescriber is requesting benefit information for a 30 day supply of LIPITOR 10MG TABLET for patient John Smith. The pharmacy is identified with the NCPDPID 1234567.

This PMBC Request example ties to all responses included in this section.
