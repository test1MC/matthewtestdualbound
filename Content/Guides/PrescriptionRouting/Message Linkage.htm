Message Linkage

Customers must follow all message linkage as described in the table below. Four key fields are used to link messages together, as defined below. This linkage allows a software system to tie one message to another. For example, on a RefillRequest the prescribing system needs to find the original NewRx to which the pharmacy is referring. This linkage should be done in an automated manner.

MessageID - The unique number assigned to each message.

RelatesToMessageID –The <MessageID> of the message being responded to.

RxRefereneNumber – The prescription number assigned by pharmacy system.

PrescriberOrderNumber – The prescription number created by the prescribing system. For diagram below this is called PON.
