Troubleshooting/FAQ

Errors Sending PAInitiationRequest Messages: Ensure that the SPI for the logged in user has been granted an ePA service level within the Surescripts Provider Directory. Additionally, ensure that the PBM/payer has been added as an allowed recipient within the user’s Edge Test Tool configuration. Allowed recipients’ PayerIds are only able to be added by Surescripts administrator users.

Schema Errors: Ensure that you are using the NCPDP SCRIPT v2013101 schema for ePA messaging. Ensure that fields are in the order defined within the schema and that they conform to the listed datatypes.

Accelerator Launch Errors: If you receive an error when attempting to launch the Accelerator from within the Edge Test Tool, please contact Surescripts Customer Support.

Edge Test Tool Launch Errors: If users cannot see the Edge Test Tool login screen, they must ensure that they are launching from a white-listed IP. Contact Surescripts Customer Support to identify which IPs have been white-listed for Staging environment or to add additional IPs.
