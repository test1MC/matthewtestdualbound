Initial Set Up

Pre-Log In Setup

Users must be set up with a Surescripts Admin Console username and password in order to access the Edge Test Tool. Once set up with Admin Console credentials, users can log into the Edge Test Tool for the purposes of testing ePA messaging. The Surescripts CompletEPA® Accelerator is also available to launch within the Edge Test Tool for the purposes of interacting with ePA messages (question sets, attachments, etc.).

For issues with logging in or obtaining an Admin Console username and password, please contact Surescripts Support or your Surescripts Account Manager.

First Time Log In

The first time a user logs in they are presented with a screen that allows them to generate a test NPI and SPI for purposes of sending PAInitiationRequests within the Edge Test Tool. Users are responsible for generating their own XML PAInitiationRequest messages or using/editing a template provided by Surescripts.

Fill out all of the required fields and click the Generate Test NPI button to assign an NPI/SPI to the user account. Once created, the user is unable to be edit their information without administrator assistance.
