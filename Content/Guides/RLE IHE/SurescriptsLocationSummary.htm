Surescripts Location Summary CDA

Each location is listed in multiple elements within the Location Summary CDA.

  1. HTML table data within the structuredBody is typically viewed by users with a stylesheet applied within the EHR.
  2. Each location is also included as an XML encounter entry with discrete data elements.
    1. Encounter entries are available to be parsed by a system for use within the EHR.
    2. When available, data includes encounter date, NPI, address, organization and provider names, phone, and specialty information as seen in the table data of the structured body.

The following table includes further explanation for key fields.

While name and address information is typically included, encounter date and NPI are the mandatory fields that will always be included. Other elements are optional and will be included when available.
