PLQ ITI-56 Response - Not a Health Data Locator

The following is an example of a PLQ Response Message sent from Surescripts to the requesting location when Surescripts never had, or no longer has information on the requested patient. The response will be a soap fault and indicate that Surescripts is ‘Not a Health Data Locator’. The requesting system should drop this Patient ID correlation and start over with an XCPD for this patient.

<soap:Envelope xmlns:soap="http://www.w3.org/2003/05/soap-envelope">

  <soap:Header>

    <Action xmlns="http://www.w3.org/2005/08/addressing">urn:ihe:iti:xcpd:2009:RespondingGateway_PortType:RespondingGateway_PatientLocationQuery:Fault:ExpectedProcessorException</Action>

    <MessageID xmlns="http://www.w3.org/2005/08/addressing">urn:uuid:094671f5-264f-4132-a86e-ceb3c01d55f1</MessageID>

    <To xmlns="http://www.w3.org/2005/08/addressing">http://www.w3.org/2005/08/addressing/anonymous</To>

    <RelatesTo xmlns="http://www.w3.org/2005/08/addressing">urn:uuid:97adff56-3721-4f4a-bdab-7920874c53f3</RelatesTo>

  </soap:Header>

  <soap:Body>

    <soap:Fault>

      <soap:Code>

        <soap:Value>soap:Sender</soap:Value>

      </soap:Code>

      <soap:Reason>

        <soap:Text xml:lang="en">Not a Health Data Locator for patient identifiers with root O&lt;Brien&gt;</soap:Text>

      </soap:Reason>

    </soap:Fault>

  </soap:Body>

</soap:Envelope>

PLQ ITI-56 Response - Error Returned Processing Request

The following is an example of a PLQ Response Message sent from Surescripts to the requesting system when an error was encountered while processing the request.

<soap:Envelope

    xmlns:soap="http://www.w3.org/2003/05/soap-envelope">

    <soap:Header>

        <Action

        xmlns="http://www.w3.org/2005/08/addressing">urn:ihe:iti:xcpd:2009:RespondingGateway_PortType:RespondingGateway_PatientLocationQuery:Fault:ExpectedProcessorException

        </Action>

        <MessageID

            xmlns="http://www.w3.org/2005/08/addressing">urn:uuid:c54b4e94-686d-4e8f-9257-430e13d6b026

        </MessageID>

        <To

            xmlns="http://www.w3.org/2005/08/addressing">http://www.w3.org/2005/08/addressing/anonymous

        </To>

        <RelatesTo

            xmlns="http://www.w3.org/2005/08/addressing">urn:uuid:6cb07a13-7c55-47a3-a299-88e5f53e7a60

        </RelatesTo>

    </soap:Header>

    <soap:Body>

        <soap:Fault>

            <soap:Code>

                <soap:Value>soap:Sender</soap:Value>

            </soap:Code>

            <soap:Reason>

                <soap:Text xml:lang="en">A patient identifier root (max length = 64 characters) must be entered.</soap:Text>

            </soap:Reason>

        </soap:Fault>

    </soap:Body>

</soap:Envelope>
