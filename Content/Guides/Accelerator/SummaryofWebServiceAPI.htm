Summary of Web Service API Endpoints

The following provides a summary of the Web Service API endpoints and the data that they provide.

API Endpoint

Description

[GET]

/uiservice/api/v0/worklist?alternateId={Patient PBM unique Id}&alternateOrganization={payer Id}&alternateQualifier=PBMUID&user={user]&pageIndex=

{pageIndex}&pageSize={pageSize}

/processes?patientId={patientId}&facility={facility}

&createdFromDate={createdFromDate}&createdThruDate={createdThruDate}

&pageIndex={pageIndex}&pageSize={pageSize}

Retrieves a paged list of processes for a given patient.

The process details include the relevant data from the ePA transactions for the current state in the workflow.

The process details include the relevant data from the SPO messages for the current state in the workflow.

[POST]

/processes

Retrieves a paged list of processes for specified initiator(s).

[POST]

/initiators/processes

Retrieves a paged list of processes for the given list of initiatorsProcessId(s).

Note: The initiatorsProcessId is the PAReferenceID as defined by the provider vendor system.

Note: The initiatorsProcessId is the SPOReferenceID as defined by the provider vendor system.

[GET]

/processes/{processId}

Retrieve the process details for a specified processId.

[GET]

/initiators/processes/{initiatorProcessId}

Retrieve the process details for a specified initiatorProcessId.

[GET]

/worklist?patientId={patientId}

&facility={facility}&user={user}

&pageIndex={pageIndex}&pageSize={pageSize}

Retrieves a paged list of the active tasks for a given patient.

[GET]

uiservice/api/v0/worklist/summary?patientId={patientId]&facility={facility}

Retrieves the count of worklist tasks for the patient.

[POST]

/worklist

Retrieves a paged list of the active tasks for a given collection of recipients (providers).

[GET]

/uiservice/api/v0/worklist/summary? alternateId={Patient PBM unique Id}&alternateOrganization={Payer Id}&alternateQualifier=PBMUID &product=mmpoc

Retrieves the count of worklist tasks for the patient.

[POST]

/uiservice/api/v0/worklist/summary?product= epa mmpoc carealert

/uiservice/api/v0/worklist/summary?product=spo

Retrieves the count of worklist tasks for the provider.

[GET]

/mmpoc/api/v0/adherence-profiles/summary?memberId={Patient PBM Unique Id}&payerId={Payer Id}

Retrieves the patient’s lowest Proportion of Days Covered ratio information.

[GET]

/mmpoc/api/v0/patient-ids?firstName={first name}&lastName= {last name}&middleName={middle name}&zipCode={zip code}&birthDate={date of birth}

Searches the Surescripts PBM/payer MPI and retrieves the pbmMemberId and payerId. Note: This is used instead of the eligibility 271 in workflows that are not associated with a patient visit/encounter.

[PUT]

/cancel/processes/{processId}

Initiates cancel for a process based on the processesId.

[PUT]

/cancel/initiators/processes/{initiatorProcessId}

Initiates cancel for a process based on the initiatorProcessesId.
