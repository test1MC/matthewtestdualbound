SRC-020

Reference Number

<SourceReference>

an..35

Prescription Number associated to medication history record.

If the submitter chooses to send this field and SRC-010-01 source qualifier is “P2”, then it must be populated with the pharmacy prescription number. Otherwise, the field is not sent.

If the source is the prescriber, this field is not sent.

If the submitter chooses to send this field and the source is the payer, this field contains the pharmacy’s prescription number from the payer system from claims processing.

DRU-080-01 contains the Prescriber’s Order Number.

C

SRC-030

Fill Number

<FillNumber>

n..2

Defines the dispensing episode as an initial fill or an authorized refill.

If submitter is pharmacy (SRC-010-01 value is P2) and chooses to send this field, the field should contain the fill number from the pharmacy.

If Source Qualifier value is “PC” (Prescriber), this field is not sent.

If submitter is payer (SRC-010-01 value is PY and chooses to send this field, the field should contain pharmacy’s fill number from the payer system from claims processing

Values:

00 = Initial fill

01= First refill

02 = Second refill, etc.

Allowed values 00 thru 99
