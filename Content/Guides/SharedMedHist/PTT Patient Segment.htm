PTT-070-01

Communication Number

<Number>

an..80

Phone Number shall*106M-025 be in the format 5552223333X4444 - where 555 is the area code, 2223333 is the main number and X4444 is the extension. Phone number shall*106M-028 not be populated with all zeros or other default values. This field is 80 characters long to handle email addresses. (Please note this extension is just an example. Less/more than four digits are allowed).

M

PTT-070-02

Code List Qualifier

<Qualifier>

an..3

If a communication number is sent, one occurrence must contain TE for telephone.

Receivers of 10.6 messages cannot expect this information to always be present due to translation from lower SCRIPT versions on the network.

For values, see the NCPDP External Code List (X12 DE 365).
