Medication History Example

In this example, the prescriber system is requesting historical drug information for a patient. The prescriber system has already determined the PBM Unique ID through the eligibility message. This unique number will be passed within the message.

In the examples, line breaks are used at the end of the segments for display purposes – live messages should not contain line breaks.
