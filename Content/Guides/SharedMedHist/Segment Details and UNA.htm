UNA – Service String Advice Segment

The UNA EDIFACT segment is a fixed-length segment. This segment determines the delimiters for the transaction which follows. The participant should use the values of each separator as defined below. The values defined below are in decimal representation with the Hex value in parenthesis. Please refer to section EDIFACT Dynamic Delimiters on page 1.
