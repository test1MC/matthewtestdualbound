About This Guide

This Surescripts Medication History for Reconciliation (MHR) Implementation Guide was created to assist acute and Long Term Care (LTC) facilities in developing and transferring messages needed to provide physicians treating hospitalized patients access to outpatient medication history. This product supports XML implementations.

The audience for this document includes any customer responsible for developing a system interface for these transactions. This guide describes the Surescripts medication history transaction sets and provides other information needed for their implementation in the acute and LTC settings.

Document References

This Guide is meant to support and integrate with the NCPDP standard schemas, implementation guides and best practices where applicable. It does not reproduce the base standard in its entirety.

This Guide includes data elements only where Surescripts has specific requirements or further explains the field usage. Refer to the SS_SCRIPT_XML_10_6MU.xsd XML Schema for a complete list of fields.
