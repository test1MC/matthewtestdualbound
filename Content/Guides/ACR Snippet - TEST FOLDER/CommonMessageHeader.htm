Common Message Header

This section only applies to customers using XML format. The common message header is not used for customers using the EDIFACT format. Surescripts uses a common message header for all messages.

For this implementation the digital signature is not used.

In the following diagram, blocks that are solid are mandatory, whereas blocks that have dotted lines are conditional.
